## [anmoldeep.app](https://anmoldeep.vercel.app/)

- Next.Js
- Framer Motion
- Tailwind CSS
- Resend
- Dev.to API
- Vercel Deployment

## Getting Started

Install dependencies

```bash
npm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
